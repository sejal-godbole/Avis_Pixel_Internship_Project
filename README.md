# Avis Pixel - School ERP System

 A modern web-based Avis Pixel ERP built with Next.js and TypeScript. This project helps manage students, teachers, classes, grades, attendance, fees, subjects, timetables, and more.

## Features
- Student, Teacher, and Class management
- Attendance tracking
- Grade and subject management
- Timetable scheduling
- Fee management
- Settings and dashboard
- Responsive UI with reusable components

## Project Structure
```
app/           # Main application pages and layouts
components/    # Reusable UI and feature components
hooks/         # Custom React hooks
lib/           # Utility functions and local storage/db helpers
public/        # Static assets
scripts/       # Database schema and scripts
styles/        # Global styles
```

## Getting Started
1. **Install dependencies:**
   ```sh
   pnpm install
   ```
2. **Run the development server:**
   ```sh
   pnpm dev
   ```
3. **Open in browser:**
   Visit `http://localhost:3000`

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- PNPM

## Folder Overview
- `app/`: Main pages and layouts
- `components/`: UI and feature components
- `hooks/`: Custom hooks
- `lib/`: Utilities and types
- `public/`: Static files
